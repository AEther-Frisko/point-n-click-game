# Point & Click Adventure Game
This is a small-scale point & click style game, which I am creating for personal learning/exploration purposes.
I intend for this to eventually be a complete game, but as of writing this it is very barebones. Not much to say just yet.

## The File Structure
I'm experimenting with organizing the structure into three main folders:
- Entities: "things" in the game, like items
- Systems: The more logic-heavy parts, like the interaction system
- Shared: Things that are related to both (aka I wasn't sure where to put them)

There are sub-folders for more specific things as well. I'm not sure how effective this will be, so we'll see how that goes.
