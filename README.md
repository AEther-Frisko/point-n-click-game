# Point & Click Adventure Game
This is a small-scale point & click style game, which I am creating for personal learning/exploration purposes.
I intend for this to eventually be a complete game, but as of writing this it is very barebones. Not much to say just yet.

## Testing the Game
I intend to include an `.exe` build of the game for testing purposes. Currently, I only have a Windows build.

If you wish to test it:
	1. Download the contents of the `builds` folder
	2. Run the `Point n Click Game.exe` file
	3. If you want to see console output, run the `Point n Click Game.console.exe` file

Let me know if there's any issues with this, I've only tested it on my own computer so far!
